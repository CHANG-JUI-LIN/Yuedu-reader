# EPUB Compatibility Checklist

Use this checklist when testing or reporting EPUB rendering issues.

## Basic metadata
- EPUB version: EPUB 2 / EPUB 3
- Layout: reflowable / fixed-layout
- Writing mode: horizontal / vertical-rl
- Navigation: toc.ncx / nav.xhtml / both

## Rendering areas
- Paragraph layout
- Headings
- Margins and padding
- Borders and backgrounds
- Images
- SVG
- Ruby
- Links
- Footnotes
- CJK vertical writing
- Table of contents
- Page breaks
- Fixed-layout viewport

## Debug information
- Book title
- Chapter or page
- Screenshot from Yuedu Reader
- Screenshot from Apple Books if available
- Whether the issue reproduces after changing font size
- Whether the issue reproduces in scroll mode or paged mode

## Minimal sample
If possible, create a small EPUB that reproduces the issue.
Do not upload copyrighted books publicly.
